# Girls on Firewalls – Networking Labs

Hands-on networking labs focused on real-world enterprise environments using Cisco Packet Tracer.

## What I Did

- Configured Layer 2 and Layer 3 networks
- Created and managed VLANs for network segmentation
- Implemented inter-VLAN routing using SVI
- Configured static and dynamic routing (BGP)
- Built multi-subnet network topologies
- Validated connectivity using ping and CLI commands

## Tools & Technologies

- Cisco Packet Tracer
- Cisco IOS CLI

## Skills Gained

- Network Security Fundamentals
- Routing & Switching
- VLAN Configuration
- TCP/IP Networking
- Network Troubleshooting

## Labs Included

- Lab 1: Switching (L2)
- Lab 2: Switching (L3)
- Lab 3: Routing Basics
- Lab 4: Dynamic Routing (BGP)
- Lab 5: Network Design Hackathon

## About This Project

This project demonstrates practical experience in networking and security concepts, including real-world configurations of switches, routers, and network protocols.



## Project Structure

- lab1-switching → Layer 2 networking fundamentals
- lab2-l3-switching → Inter-VLAN routing and Layer 3 switching
- lab3-routing → Static routing and network connectivity
- lab4-bgp → Dynamic routing using BGP
- lab5-hackathon → Network design and high availability concepts
